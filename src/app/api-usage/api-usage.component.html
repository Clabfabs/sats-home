API Usage

To use SATS in your own code, you can either use sats-core to generate auction instances or sats-opt, which extens sats-core, to solve the winner determination problems.

Getting started

Both components are deployed on Maven Central, so with Maven, including sats-core is as easy as:


    <dependency>
      <groupId>org.spectrumauctions</groupId>
      <artifactId>sats-core</artifactId>
      <version>{{getCoreVersion() || 'Loading...'}}</version>
    </dependency>

Or, for sats-opt:


      <dependency>
        <groupId>org.spectrumauctions</groupId>
        <artifactId>sats-opt</artifactId>
        <version>{{getOptVersion() || 'Loading...'}}</version>
      </dependency>

This is enough information for Maven to download the dependency automatically.

Alternative without maven:

Both APIs are available as precompiled jar-files on Github (see Home). You can include them in your project in directly.
