SATS

A Universal Spectrum Auction Test Suite

Welcome!

On this page you will find all information about SATS, the Spectrum Action Test Suite. SATS is based on the Paper "SATS: A Universal Spectrum Auction Test Suite", published at AAMAS 2017. It can be downloaded here.

SATS consists of three components:

SATS-WEB

For basic use of SATS we provide a web application. No installation is required - everything in done in your browser.

SATS-CORE and SATS-OPT

For advanced use of SATS we provide a Java library which allows scalable and configurable experimental setups.

SATS-CLT

For basic but scalable use of SATS we provide a command line tool (CLT) to generate value files in your command line.
